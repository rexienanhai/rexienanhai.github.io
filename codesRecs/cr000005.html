Instruction of ESSI-RSM's installation

Record Date: April 22, 2019



1. First of all, you should download the ArcEngine 10.2 (Named as MAPS_Dependency.7z)。

2. Unzip the MAPS_Dependency.7z，Then install the ArcEngine10.2。 


How to install the ArcEngine 10.2

1. Firstly, the ArcGIS License Manager should be installed.


2. Replace the files of service.txt and ARCGIS.exe

3. Start the service


4. Install the ArcGIS Engine Runtime



How to install the ESSI-RSM

1. Download the ESSI-RSM software package

2. Then install it. Attention please: Don't install it in the 'C' disk.

3. The software need to register using register code, and you can send the machine code to author and get the register code!
